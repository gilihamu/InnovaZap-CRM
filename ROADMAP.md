# InnovaZap CRM Roadmap

## Phase 1 - MVP (Weeks 1-6)
- [IZP-1] Docker Compose dev environment
- [IZP-2] InnovaZap.Shared DTOs and contracts
- [IZP-3] Gateway with JWT auth + multitenant
- [IZP-4] CRM Service (Leads, Pipeline, Tasks)
- [IZP-5] Messaging Service (WhatsApp webhook + inbox)
- [IZP-6] Chatbot flow engine
- [IZP-7] Frontend setup (React + Vite + TailwindCSS)
- [IZP-8] LeadsPage with table and drawer
- [IZP-9] PipelineBoard kanban
- [IZP-10] InboxPage with conversations
- [IZP-11] EF Core migrations
- [IZP-12] Solution scaffolding

## Phase 2 - Automation (Weeks 7-10)
- Follow-up automático via RabbitMQ
- Resposta fora do horário
- Qualificação automática de leads (hot/warm/cold)
- Automation engine (cron + event triggers)

## Phase 3 - Analytics + AI (Weeks 11-14)
- Dashboard executivo (KPIs, funnel, conversion)
- AI lead classification (Anthropic API)
- AI response suggestions
- PDF/Excel report export

## Phase 4 - Scale (Weeks 15+)
- Multi-agent queue distribution
- Flow template marketplace
- External webhooks
- Kubernetes deployment + CI/CD
